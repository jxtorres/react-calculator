Calculator Modulus Game
---


Game to reach Zero with fewest moves.

GAME INSTRUCTIONS:
1. Press AC.
2. Start Pressing Number keys to reduce the total.
3. Reach Zero.

Calculator Logic Forked from ahfarmer/calculator. 



Try It
---

---
